🚀 Quick Start

  1. Install Python 3.8+ if not already installed
  2. Install dependencies: pip install -r requirements.txt
  3. Run the application: streamlit run phishing_gui.py
  4. Open your browser to http://localhost:8501

Alternative: Double-click start_web_app.bat (Windows) or run python setup_and_run.py

📋 System Requirements

  • Python 3.8 or higher
  • 8GB RAM recommended
  • Internet connection for real-time analysis
  • Modern web browser (Chrome, Firefox, Safari, Edge)

🔧 Troubleshooting

  • Port already in use: Try streamlit run phishing_gui.py --server.port 8502
  • Missing packages: Run pip install -r requirements.txt
  • Permission errors: Try running as administrator (Windows) or with sudo (Linux/Mac)
  • File not found: Ensure you're in the project directory
